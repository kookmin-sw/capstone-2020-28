### 1. 프로젝트 개요
<p> IT가 발전하면서 보안의 필요성 또한 증가하고 있다. 보안을 하기 위해 필요한 것은 로그를 확인하는 것 이다. 그러나 로그의 양은 방대하고 모든 로그를 다 보는 것은 사실상 어려운 일이다. 그리하여 본 프로젝트에서는 로그를 시각화하여 현재 상태를 직관적으로 보여주는 로그 통계 리포트를 개발하려고 한다.
